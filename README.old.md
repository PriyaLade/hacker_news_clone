### How to quickly set up

Clone this repository

```sh
git clone git@github.com:clivemchd/hacker_news_clone.git
```

Install the dependencies and devDependencies and start the server.

```sh
$ cd hacker_news_clone
$ npm install
```

To start server...

**Linux, macOS (Bash)**

```ssh
> npm start
```

```ssh
> npm start
```
